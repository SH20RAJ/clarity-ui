# Introduction

Clarity UI is a collection of re-usable components that you can copy and paste into your apps.

## Features

- **Customizable**: Built on top of Radix UI and Tailwind CSS.
- **Accessible**: Integrated with Radix UI primitives.
- **Open Source**: MIT licensed.

## FAQ

**Is this a component library?**
Yes and no. It's a collection of components that you can copy and paste into your apps. You don't install it as a dependency.

**Can I use this in my project?**
Yes. Free of charge for personal and commercial projects.
